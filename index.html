In-text link

CSS selector:

.element1 a

Sample code:

<a href="https://www.instagram.com/hive.cafe/">@hive.cafe</a>

@hive.cafe

Navigation Link

CSS selector:

.button-nav

Sample code:

<a href="#" class="button">Default Button</a>

Rendered element:
